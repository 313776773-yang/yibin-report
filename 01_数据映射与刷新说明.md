# 数据映射与刷新说明

## 一、数据链路

`数据源Excel → 网页内转换或一键脚本 → data/dashboard_data.json → HTML六个页面联动刷新`

数据源文件：

`data_source/宜宾商都AI经营决策中枢_数据确认与演示模板_v1.0.xlsx`

## 二、更新步骤

1. 在Excel中维护基础数据表并保存。蓝色字体区域为输入区；不要修改字段名称、唯一编号和表间关联键。
2. 确认Excel中“24_模型检查”的MODEL STATUS为PASS。
3. 推荐直接打开看板，点击右上角“导入Excel”。浏览器在本机解析工作簿并立即刷新，不上传文件。
4. 需要同步生成固定JSON和单文件演示版时，双击 `一键更新数据并打开看板.command`。
5. 如需命令行批量运行，可在套件根目录执行：

```bash
python3 scripts/export_dashboard_data.py \
  --input data_source/宜宾商都AI经营决策中枢_数据确认与演示模板_v1.0.xlsx \
  --output data/dashboard_data.json \
  --js-output assets/default-data.js
```

6. 重新打开或刷新 `index.html`。如果不替换文件，也可点击“导入JSON”选择新快照。

## 四、真实AI接入位置

正式系统建议采用三层分工：

1. 指标引擎：负责统一口径、勾稽、覆盖率与基础计算。
2. 预测/异常模型：负责趋势预测、异常识别、影响测算和方案模拟。
3. 大模型与知识库：读取结构化指标、运营手册、授权矩阵和历史案例，生成“事实—推断—方案—可信度—授权边界”决策卡。

大模型不承担财务计算，也不得绕过数据边界生成不存在的经营事实。
